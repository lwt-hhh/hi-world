# const   有什么用途：





- 可以定义const   常量

- const  可以修饰函数的参数、返回值，甚至函数

​       的定义体。被const  修饰的东西都受到强制保护，可

​      以预防意外的变动，能提高程序的健壮性。

 

